# text_overlay.md

Claudeがauto_layout.pyで画像に合成した文字内容と、その配置・スタイルの記録。

canva_text.md（Canva用）を置き換えるファイル（Ver3.0）。Canvaは使用しない。

Claude（レビュー）が文字内容を検証するための記録であり、次回セッションのClaudeが同じ調整を繰り返さずに済むための引き継ぎメモを兼ねる。

Claudeは投稿ごとにこのファイルの中身を全文書き換える（下記はテンプレート）。
