# image_request.md（Ver3.2テンプレート・見本＋差分方式）

Ver3.1までは、世界観・ブランド・文字ゾーン設計・チャート仕様・出力仕様など全部を毎回フルで書いていたが、
「見本を完全再現して」と「今回はここを変えて」が同じ分量で並んでしまい、画像生成担当AIにとって
どこまで固定でどこから変更していいかの判断が難しい、という指摘を受けた（2026-07-11）。

Ver3.2より、**見本画像を1枚添付し、「今回変更する点」だけを短く書く**形式に変更する。
世界観・ブランド・文字ゾーン設計・出力仕様の説明は、これ以降は毎回書かない（見本画像に任せる）。

---

## このメッセージの使い方

1. 該当する見本画像（Brand_Reference/01_Bestより、カットの型に合うもの）を1枚添付する
2. 下記のテンプレート本文をコピペする
3. 「今回変更する点」だけを埋める

---

## テンプレート本文

```
添付画像を完全に再現してください。

【完全再現する部分（変更しない）】
・世界観、キャラクター、クマ、背景、配色、線画タッチ、フォント選定
・バッジ・吹き出し・ボックスなどの「形」そのもの
・文字は一切描かない（バッジ内・吹き出し内・チャートラベル・チェックリスト部分はすべて空欄のまま）

【今回変更する点】
（ここに、今回のカットで見本と変える内容だけを箇条書きで書く。例：
・チャートは1枚のみ
・チェックリストの丸を3つ均等配置
・人物はスマホを見せるポーズ
・くまはOKサイン
・チャートの中身：25日線が左で下、中央で交差、右で上（ゴールデンクロス）、
　交差点で出来高（棒グラフ）が増加、交差する足は陽線）

【出力サイズ】
1080×1350px（リールの場合は1080×1920px）、PNG、sRGB、トリミング禁止。
このメッセージは1カットのみの依頼です。他のカットと合成しないでください。
```

## 線が交差するチャートを含むカットの場合（重要）

ゴールデンクロス・デッドクロスなど、2本の移動平均線が交差するチャートを含むカットは、
文章の指示だけでは「交差後に線が消える」不具合が起きることが確認されている。

必ず以下の手順を踏む。

1. `charts/chart_generator.py`で、そのカットに合う参考チャート画像を先に生成する
   （例：`python3 chart_generator.py --pattern dead_cross --out chart_ref.png`）
2. ブランド見本画像（qa_line_position_01等）と、生成した参考チャート画像の**2枚**を添付する
3. 本文に「添付1（ブランド見本）はレイアウト・キャラクター等を再現、添付2（参考チャート）はチャート部分をそのまま再現」
   と役割を分けて明記する

---

過去の「線の向き」テーマ制作時の見本だが、レイアウトの型は共通して使える。

| カットの型 | 見本ファイル名 |
|---|---|
| 導入・2チャート比較＋吹き出し | qa_line_position_01 |
| チェックリスト型 | qa_line_position_02 |
| 凡例型 | qa_line_position_03 |
| 6コマ漫画型 | aa_line_position_01（推定。要目視確認） |
| リール（問題／答え／解説） | rl_line_position_01／02／03 |

※ aa_line_position_01については実際に中身を目視確認していないため、使う前に一度開いて型が合っているか確認すること。
