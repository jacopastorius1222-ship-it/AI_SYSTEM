## 1. 各AIの役割

### ChatGPT｜ライター

**担当：投稿の下書き作成**

- 人間からテーマを受け取り、投稿の初稿を作成する
- 参照ルール：PROJECT_RULE.md
- 出力物：タイトル・キャプション・ハッシュタグの初稿

---

### Claude｜Instagram編集長

**担当：コンテンツ制作・レビュー**

制作工程（投稿作成モード）と評価工程（レビューモード）の2つを担当する。

#### 投稿作成モード

- テーマ選定・タイトル・キャプション・ハッシュタグ・画像設計書・リール構成・セルフレビューを担当する
- 参照ルール：PROJECT_RULE.md（最優先）・CONTENT_RULE.md

#### レビューモード

- ChatGPTまたはCodexが作成した投稿をPROJECT_RULE.mdだけを基準に採点する
- 良い点は不要。改善点のみ指摘する
- 改善案は必ず3案以上出す
- 100点満点で採点し、★評価と改善優先順位を出力する
- 参照ルール：PROJECT_RULE.md（のみ）

#### Claudeが担当しないこと

- 画像の生成・出力
- 動画の編集・書き出し

---

### Codex｜画像クリエイター

**担当：画像の生成**

- Claudeが出力した画像設計書をもとに画像を生成する
- 参照ルール：PROJECT_RULE.md（セクション11 共通イラストルール）・Claudeが出力した画像設計書

#### Codexが担当しないこと

- テーマ選定・キャプション・ハッシュタグの作成
- レビュー・採点

## 2. ファイルの流れ

```
人間
│
│ テーマを入力
▼
ChatGPT
│
│ 投稿初稿を作成（タイトル・キャプション・ハッシュタグ）
│ 参照：PROJECT_RULE.md
▼
Claude（レビューモード）
│
│ 初稿をPROJECT_RULE.mdで採点
│ 改善点・改善案（3案以上）・100点採点・★評価・優先順位を出力
▼
人間
│
│ 改善内容を確認・承認
▼
Claude（投稿作成モード）※必要な場合
│
│ 修正版を作成（タイトル・キャプション・ハッシュタグ・画像設計書・リール構成）
│ 参照：PROJECT_RULE.md・CONTENT_RULE.md
▼
Codex
│
│ 画像設計書をもとに画像を生成
│ 参照：PROJECT_RULE.md セクション11・画像設計書
▼
人間
│
│ 最終確認・投稿
▼
完成
```

## 4. 人間が行うこと

- テーマの入力
- レビュー結果の確認と承認
- 最終投稿の確認と公開

> 人間の作業はこの3点のみ。それ以外はAIが担当する。
