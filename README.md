# AI_SYSTEM

このリポジトリは、株初心者向けInstagram運用のAIルール保管庫です。

## 最優先ルール

PROJECT_RULE.md

Instagram投稿を作成・レビューする場合は、必ず最初に PROJECT_RULE.md を確認すること。

## 重要

MASTER_RULE.md、INSTAGRAM_RULE.md、REVIEW_CHECKLIST.md は参考ファイルです。

現在の実行ルールは PROJECT_RULE.md に統合済みです。

ルールが競合した場合は PROJECT_RULE.md を最優先します。

PROJECT_RULE.md が見つからない場合は、作業を開始しないでください。
