# 絶対に再現すること

参考画像と同じ

・キャラクター

・クマ

・色

・雰囲気

・余白

・タイトル

・文字サイズ

・イラスト品質

・世界観

を維持する。

新しいデザインを提案してはいけない。

ブランド変更は禁止。

改善するときも
参考画像の世界観を崩してはいけない。

参考画像を100%再現することを最優先とする。
